# NotesDesk

macOS 自用工具：在桌面**浮窗**中只读显示「备忘录」指定文件夹里**默认排序第一条**笔记的全文，并在备忘录修改后**近实时**自动同步。

> **项目总览**：[NotesDesk README](../README.md)

## 文档

- [需求说明](docs/需求说明.md)
- [架构与开发文档](docs/架构.md)

## 功能

- 菜单栏（`note.text`）选择备忘录文件夹
- 桌面浮窗显示第一条笔记（标题 + 正文，**可滚动**）
- FSEvents 监听 Notes 数据目录，**约 1 秒内**自动刷新
- 双击浮窗在系统备忘录中打开该笔记
- 可调透明度、字号、窗口位置

## 环境

- macOS 13+
- Xcode Command Line Tools 或 Xcode 15+

## 构建与运行

```bash
cd notes-desk-panel
./run.sh
```

## 首次权限

1. **自动化**：**系统设置 → 隐私与安全性 → 自动化**，允许 NotesDesk 控制「备忘录」
2. 菜单栏 `note.text` 图标 → 选择文件夹

## 与小组件版对比

| | 浮窗版（本项目） | [小组件版](../notes-desk-widget/) |
|--|------------------|-----------------------------------|
| 展示 | 可滚动全文 | 固定尺寸，大组件约 24 行 |
| 进程 | **单 App**（菜单栏 + 浮窗一体） | Host + Widget Extension |
| 自动更新 | FSEvents，近实时 | Host 轮询约 5s |
| 推荐场景 | 日常阅读全文 | 原生桌面小组件外观 |
