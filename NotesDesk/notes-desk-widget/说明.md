# NotesDesk Widget

WidgetKit 版备忘录**桌面小组件**：与 [`notes-desk-panel`](../notes-desk-panel/) 浮窗版共用同一套业务需求，以系统原生小组件形态展示指定文件夹第一条备忘录。

> **项目总览**：[NotesDesk README](../README.md)

## 架构概要

- **NotesDeskWidget Host**（菜单栏 `note.text`）：AppleScript 读备忘录 → 写入 localhost / Keychain / App Group → 刷新 Widget
- **NotesDeskWidgetExtension**：从 Host 提供的快照读取并展示；点击跳转备忘录

**Host 必须保持运行**——Widget 扩展沙盒无法直接读备忘录。详见 [架构文档](docs/架构.md)。

## 快速开始

### 1. 签名并运行 Host（必需）

**环境**：macOS 14+、Xcode 15+、`brew install xcodegen`

```bash
brew install xcodegen   # 若未安装
cd notes-desk-widget
xcodegen generate
open NotesDeskWidget.xcodeproj
```

Xcode → 两个 Target 配置 **Signing Team**（`project.yml` 中可填写 Team ID）与 **App Group** `group.com.notesdesk.widget` → **⌘R**

授予 **自动化 → 备忘录** 权限，菜单栏选择文件夹 → **「立即同步并刷新 Widget」**。

> `./run-host.sh` **不能**联调 Widget（无 App Group）。必须用 Xcode ⌘R。

### 2. 添加桌面小组件

1. 桌面 Control + 点击 → **编辑小组件**
2. 搜索 **NotesDesk** → 拖到桌面（建议**大尺寸**）
3. Widget 跟随 Host 所选文件夹，**无需在小组件里再选**

### 3. 调试

```bash
./debug-check.sh
curl http://127.0.0.1:19876/snapshot   # 应返回 JSON 快照
```


## 与浮窗版对比

| | 浮窗版 | 小组件版（本项目） |
|--|--------|---------------------|
| 展示 | 可滚动全文 | 固定尺寸，约 24 行 |
| 进程 | 单 App | Host + Extension |
| 自动更新 | FSEvents，~1s | 轮询 ~5s，Widget 时间线 ~10s |
| 构建 | `./run.sh` | Xcode 签名 ⌘R |
| 推荐 | 日常全文阅读 | 原生小组件外观 |

## App Group

`group.com.notesdesk.widget` — Host 与 Widget Extension 须一致。
