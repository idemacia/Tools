# 更新日志

## [1.0.0] - 2025-02

### 新增

- 开源简化版，基于阿里云 IoT
- 智能灯控制页面
- 智能插座控制页面
- 设备服务 `device-service`：getDeviceStatus、controlDevice、invokeService 等
- 阿里云连接器 `aliyun-connector`：支持自定义 Identifier
- 配置模板 `iot-config.js`
- 开发文档：ARCHITECTURE.md、EXTENDING.md、CONTRIBUTING.md

### 说明

本项目从 MuCloud 智能家居项目简化而来，仅保留阿里云 IoT 连接能力及 smartled、socket 两个设备页面，便于二次开发和开源使用。
