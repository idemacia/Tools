# 扩展开发指南

本文档说明如何为本项目新增设备类型页面，便于第三方开发者快速扩展。

## 一、新增设备页面流程

### 1. 配置设备名称

在 `config/iot-config.js` 中增加对应设备的 DeviceName：

```javascript
deviceNameXXX: wx.getStorageSync('deviceNameXXX') || 'YOUR_DEVICE_NAME',
```

### 2. 注册页面

在 `app.json` 的 `pages` 数组中添加新页面路径：

```json
"pages/yourdevice/yourdevice"
```

### 3. 添加首页入口

在 `pages/index/index.js` 的 `devices` 数组中添加：

```javascript
{
  id: 'yourdevice',
  name: '您的设备',
  icon: '/images/icons/yourdevice.png',
  path: '/pages/yourdevice/yourdevice'
}
```

### 4. 创建页面文件

复制 `pages/smartled/` 或 `pages/socket/` 作为模板，创建以下文件：

- `pages/yourdevice/yourdevice.js`
- `pages/yourdevice/yourdevice.wxml`
- `pages/yourdevice/yourdevice.wxss`
- `pages/yourdevice/yourdevice.json`

### 5. 页面逻辑改造

在 `yourdevice.js` 中：

1. 读取设备配置：
```javascript
const deviceSn = iotConfig.deviceNameXXX || iotConfig.deviceName || wx.getStorageSync('deviceNameXXX');
```

2. 获取状态：
```javascript
deviceService.getDeviceStatus(deviceId)
  .then(status => {
    // status 为物模型 Identifier → 值的映射；智能灯常见 power、brightness、hue 等
    this.setData({ ... });
  });
```

3. 下发控制：
```javascript
// 方式一：使用已有方法（若有）
deviceService.toggleSmartPlug(deviceId, true);

// 方式二：controlDevice（内部按命令字段分支）
// 智能灯页面习惯传 ledswitch / lightBrightness 等，会映射为 power / brightness 后 SetDeviceProperty
deviceService.controlDevice(deviceId, { ledswitch: 1, lightBrightness: 80 });

// 方式三：invokeService（自定义服务，默认 Identifier 为 set，可按产品修改）
deviceService.invokeService(deviceId, { arg1: 1 }, 'your_service_identifier');
```

## 二、阿里云物模型说明

新增设备前，需在阿里云 IoT 控制台为产品定义物模型（TSL）：

- **读属性**：`QueryDevicePropertyStatus`，`getDeviceStatus` 经 `parseDeviceData` 解析（bool/数值会做类型处理）
- **写属性（智能灯默认）**：`SetDeviceProperty`，`Items` 为 JSON 字符串，字段名与物模型 Identifier 一致
- **调服务**：`InvokeThingService`，需指定 `Identifier`；插座控制当前走此路径

若产品与默认不一致，可改 `config/iot-config.js` 中的 `smartledUseInvokeThingService`、`socketThingServiceIdentifier` 等，或直接调用 `invokeService`。

## 三、device-service API 速查

| 方法 | 说明 |
|------|------|
| `getDeviceStatus(deviceId)` | 获取设备属性状态 |
| `controlDevice(deviceId, command)` | 智能灯→`SetDeviceProperty`（默认）；插座→`InvokeThingService` |
| `invokeService(deviceId, args, identifier)` | 调用指定服务（Identifier 默认 `set`） |
| `toggleSmartPlug(deviceId, state)` | 插座开关 |
| `setLightBrightness(deviceId, brightness)` | 灯光亮度（内部走 controlDevice） |
| `controlPump(deviceId, pumpNo, state)` | 水泵（`SetDeviceProperty`） |
| `checkDeviceOnline(deviceId)` | 检查在线状态 |

## 四、添加设备图标

将设备图标放入 `images/icons/`，命名为 `yourdevice_white.png` 或类似，在 index 和页面中引用。
